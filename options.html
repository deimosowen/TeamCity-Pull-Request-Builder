TeamCity Pull Request Builder

Options

Configuration Parameters:

  • BaseUrl: (Required) The URL of your TeamCity server. Example: https://yourteamcityserver.com/.
  • Username: (Required) The username for your TeamCity server.
  • Password: (Required) The password for your TeamCity server.
  • Repository: (Required) An object where each property represents a repository, and the value is an array of build configuration objects. Replace "MyRepo" with the name of your repository.
  • BuildType: (Required) The identifier of the build type in TeamCity.
  • Name: (Required) The name that will be displayed in the plugin interface.
  • Group: (Optional) The group to which this build configuration belongs. If specified, the build configurations will be grouped under this name in the plugin interface.
  • Order: (Optional) The sequential number to determine the display order of the build configurations within a group.
