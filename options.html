TeamCity Pull Request Builder

Options

Description

1. BaseUrl (Обязательный параметр):

Адрес сервера TeamCity. Замените значение "https://ci.parcsis.org/" на адрес вашего сервера TeamCity.

2. Repository (Обязательный параметр):

Словарь, содержащий информацию о репозиториях, на которых будет доступна вкладка "Build in TeamCity". Каждый ключ в этом словаре представляет собой название репозитория. В приведенном примере ключ - "CasePro".

2.1 CasePro:

Массив, содержащий объекты с информацией о конфигурациях сборки TeamCity для репозитория "CasePro". Замените "CasePro" на название вашего репозитория и добавьте соответствующие объекты с информацией о конфигурациях сборки.

2.1.1 BuildType (Обязательный параметр):

Наименование конфигурации в TeamCity.

2.1.2 Name (Обязательный параметр):

Отображаемое имя для конфигурации сборки.
